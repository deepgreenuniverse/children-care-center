# 小树苗看护 - 儿童疾病智能护理提醒工具

## 一句话介绍

孩子生病时，家长最怕忘记喂水、量体温。小树苗看护根据疾病类型自动生成全套护理提醒方案，到点就提醒，帮你扛过每一次手足无措的看护夜。

## 解决什么问题

儿童患病期间的护理高度依赖"按时执行"——手足口病要半小时喝一次水、肺炎要按时雾化拍背、腹泻要频繁补液防脱水。但家长在焦虑和疲惫中极易遗漏。小树苗看护把每种常见疾病的护理节奏内置好，一键启动，按时推送，家长只需要"听到提醒 -> 执行 -> 点完成"三步走。

## 核心功能

### 内置7种常见儿童疾病方案

| 疾病 | 喝水 | 如厕 | 体温 | 额外护理 |
|------|:--:|:--:|:--:|------|
| 手足口病 | 30分钟 | 2小时 | 4小时 | 口腔护理 + 皮疹观察 |
| 流感/感冒 | 1小时 | 3小时 | 4小时 | 吃药提醒 |
| 腹泻 | 20分钟 | 按需 | 6小时 | 补液盐提醒 |
| 咽峡炎 | 30分钟 | 2小时 | 4小时 | 漱口 + 冷流质饮食 |
| 肺炎 | 40分钟 | 2小时 | 3小时 | 雾化 + 拍背排痰 |
| 幼儿急疹 | 1小时 | 3小时 | 3小时 | 退烧药间隔 |
| 便秘 | 1小时 | 2小时 | - | 腹部按摩 |

> 所有间隔均可自定义调整，也支持完全自定义疾病和提醒项。

### 看护中：倒计时 + 一键完成

- 圆形进度环实时显示距下次提醒还有多久
- 提醒到达时弹窗 + 提示音，支持「已完成」和「推迟5分钟」
- 今日统计面板：喝水 x 次 / 如厕 x 次 / 体温 x 次，一目了然

### 看护记录 + 体温追踪

- 每次看护自动生成日志：疾病名、时长、各项完成次数
- 体温手动录入，表格记录，方便就诊时出示

### 多端支持

- **H5版**：浏览器即开即用，无需安装，手机电脑都能跑
- **Android版**：WebView内嵌 + 系统级AlarmManager通知，App在后台也能准时弹提醒，支持通知栏快捷操作

## 技术特点

- H5版纯HTML+CSS+JS，零依赖，单文件，部署即上线
- Android版原生通知通道，不依赖App前台运行
- 数据全量存储在本地（localStorage），不上传任何服务器
- 儿童主题配色，大字体大按钮，照顾家长疲惫状态下的操作体验

## 适用场景

- 孩子确诊手足口病、流感、腹泻等常见儿科疾病后的居家护理
- 家长需要按医嘱执行高频护理动作（喝水/如厕/量体温/吃药/雾化等）
- 多家庭成员协作看护时，避免重复或遗漏
- 就诊时向医生出示完整的居家护理记录
